# **App Name**: Ministry Hub

## Core Features:

- Homepage: Display key information like service times, featured ministries, upcoming events, and a sermon preview.
- Sermon Management: Enable upload and categorization of sermons with video, audio, notes, and scripture references.
- Online Giving: Integrate with payment gateways like Paystack/Flutterwave to support one-time donations, tithes, and mission support.
- Event Calendar: Allow display of upcoming programs, annual events, and program flyers in a calendar view.
- Contact and Prayer Request: Provide a contact form and prayer request submission feature.
- AI-Powered Content Tool: Tool that uses a generative AI to create summaries and excerpts of uploaded sermons and blog posts to improve searchability and engagement. 

## Style Guidelines:

- Primary color: Deep navy blue (#2E3192), conveying trust and stability, fitting for a spiritual environment.
- Background color: Very light blue (#F0F4FF), creating a calm and inviting atmosphere.
- Accent color: Soft gold (#D4AF37), used sparingly for highlights and calls to action, evoking a sense of reverence and tradition.
- Headline font: 'Playfair', a modern serif that offers an elegant and fashionable feel for headlines.
- Body font: 'PT Sans', a humanist sans-serif providing a modern look, well-suited for body text.  Paired with 'Playfair' for headlines.
- Use clear and simple icons to represent different ministries and events.
- Implement a grid-based layout for clean and organized content presentation.
